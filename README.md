# E-Commerce API 🛒

This project is a RESTful API built for an e-commerce platform. It handles Category and Product management operations using a Node.js backend and a MySQL relational database.

## Project Structure (MVC Architecture)
- **`controllers/`**: Contains the logic for CRUD operations.
- **`routes/`**: Handles the API endpoints and routing.
- **`db.js`**: Manages the MySQL database connection.
- **`server.js`**: The main entry point of the Express application.

## Features
- **Category CRUD:** Create, Read, Update, and Delete product categories.
- **Product CRUD:** Manage products and link them to their respective categories using Foreign Keys.
- **Relational Database:** Designed with MySQL for data integrity.

## Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** MySQL
